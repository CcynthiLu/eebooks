BER curves for Differential PSK (DPSK) and Differential QPSK (DQPSK)

On the Digital Communications: Design for the Real World CD is a file MATLAB/BER_DPSK.M

This MATLAB function can be used with MATLAB version 4 or above to plot BER curves for Differential PSK (DPSK) and Differential QPSK (DQPSK).
