Answer 6.5

The Shannon–Hartley theorem (see section 2.4) can be written as:

C/B = log2[1 + Eb.C/N0.B]

Now for Eb/N0 equal to -1.2 dB, the ratio Eb/N0 = 10-1.2/10 = 0.76.

The bandwidth efficiency C/B that can be supported is thus:

C/B = log2[1 + 0.76C/B]

therefore:

C/B = 0.3 bits/second/Hz (approximately).
