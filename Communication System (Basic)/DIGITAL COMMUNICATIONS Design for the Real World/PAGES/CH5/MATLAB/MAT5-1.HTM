The spectrum for Sunde's FSK and Minimum Shift Keyed (MSK) random data streams

On the Digital Communications: Design for the Real World CD is a file MATLAB/SUNDEFSK.M

This MATLAB function can be used with MATLAB version 4 or above to plot the spectrum for Sunde's FSK and Minimum Shift Keyed (MSK) random data streams.
