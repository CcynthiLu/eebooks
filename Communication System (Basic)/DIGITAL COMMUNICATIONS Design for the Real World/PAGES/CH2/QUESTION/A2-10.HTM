Answer 2.10

Applying the Shannon–Hartley equation we obtain:

C = 3000.log2(103 + 1) = 29.9 kbps

Note: the S/N ratio must be converted from dB to a pure ratio, for example 30 dB is 103 when used in this equation.

In order to determine the number of symbol states required to support this data rate, we can apply the expression:

C = 2B.log2M  (for a baseband channel)

hence:

M = 2C/2B = 31.6 (32) signalling states required.
